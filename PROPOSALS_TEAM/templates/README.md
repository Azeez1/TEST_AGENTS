## How to Use Templates

1. **Choose the Right Template**
   - Government: For federal, state, or local government RFPs
   - Private B2B: For business-to-business proposals
   - Enterprise: For Fortune 500 and large enterprise RFPs

2. **Customize Thoroughly**
   - Never use as-is
   - Replace all placeholders with client-specific content
   - Add/remove sections based on RFP requirements
   - Adjust tone for client's industry

3. **Maintain Compliance**
   - Government: Follow exact RFP format if specified
   - Enterprise: Align with evaluation criteria
   - Private: Focus on relationship and value

4. **Save New Versions**
   - Don't overwrite templates
   - Save customized versions to appropriate outputs folder
   - Use naming convention: `ClientName_Sector_Date_Proposal.docx`
